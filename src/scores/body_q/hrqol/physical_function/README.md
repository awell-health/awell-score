# BODY-Q - Physical Function

## Introduction

The BODY-Q is a patient-reported outcome measure (PROM) developed using Rasch Measurement Theory (RMT) to evaluate outcomes in bariatric and body contouring surgery. The Physical Function scale is one of 30 independently functioning scales within the BODY-Q. It asks patients how often they had a problem with various physical activities over the past week using a 4-point response scale (1 = All of the Time to 4 = Never).

## Questions

The scale contains 7 items asking how often the patient had a problem with:

1. Getting up out of bed
2. Bending from side to side
3. Walking or getting around
4. Bending forward (e.g. to tie shoes)
5. Moderate exercise (e.g. brisk walking)
6. Going up or down stairs
7. Standing for a long time

## Calculation

Higher scores reflect a better outcome. If missing data is less than 50% of the scale's items, the mean of the completed items is inserted for missing values. The raw summed score is then converted to a Rasch-transformed score from 0 (worst) to 100 (best) using the conversion table below.
